# ClipCrew

AI social media agency for podcasters.

ClipCrew turns a long podcast episode into ranked, ready-to-post short clips using a Hermes-powered agent crew: producer, transcriber, moment scorer, clip cutters, caption writer, creator reviewer, and publisher.

## Hackathon positioning

**Track:** AI as Agency  
**Core promise:** Drop a podcast episode; an agent crew ships social-ready clips with visible handoffs, scores, approval, and publish receipts.

This is not just a podcast clipping tool. It is an AI agency that replaces the workflow of a podcast social media team.

## MVP flow

1. Creator uploads/pastes a podcast episode.
2. Producer agent plans the job.
3. Transcriber creates timestamped transcript.
4. Moment scorer finds candidate clips.
5. Judge panel scores hooks and rejects weak moments.
6. Cutter agents generate short vertical clips.
7. Caption agent writes social copy and subtitles.
8. Creator approval page allows approve/reject.
9. Publisher exports or posts approved clips.
10. Run log shows every agent step and output.

## Judge demo goal

Show a real episode going through the crew, then open:

- the run log / trace
- candidate clip scores
- approved clip preview
- final generated clip
- publish/export receipt
